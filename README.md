## 📌 项目简介

**优化师分身** 是一款基于 Coze + Streamlit 打造的 AI 投放策略助手，聚焦腾讯广告金融借贷行业，为优化师提供智能诊断、策略推荐、素材灵感三大核心能力。

### 核心价值
- **智能诊断**：输入账户数据，AI输出健康度评分+TOP3紧急问题+可落地建议
- **策略推荐**：基于推广目标+预算+行业，输出完整的出价/时段/素材方案
- **素材灵感**：输入产品+人群，生成创意方向+文案模板+妙思/巨量创意官方素材链接

### 适用人群
- 信息流投放优化师（3-10年经验）
- 投放主管/经理（需要团队提效工具）
- 转型AI PM的营销从业者（Demo项目参考）

## 🎯 功能说明

### 场景1：智能诊断
**输入**：账户核心指标（CTR/CVR/CPA等）
**输出**：
- 健康度评分（0-100）+ 评级（S/A/B/C/D）
- TOP3紧急问题 + 原因分析
- 可落地优化建议（具体操作步骤+预期效果）
- 行业基准对比（按件均分档）

### 场景2：策略推荐
**输入**：推广目标 + 日预算 + 行业 + 产品类型
**输出**：
- 三阶段出价策略（冷启动→稳定期→放量期）
- 盯盘节奏与成本管控时间表
- 素材方向建议 + 近期跑量案例参考
- 人群定向建议 + 关键KPI目标

### 场景3：素材灵感
**输入**：行业 + 产品类型 + 优化诉求
**输出**：
- 跑量素材公式（核心结构）
- 开场3秒钩子参考
- 5-6个拓展方向（含CTR预期/难度/生命周期）
- 行业素材趋势要点
- **妙思灵感/巨量创意官方链接**（直接可跳转参考）

## 💡 项目亮点

1. **垂直场景深耕**：聚焦腾讯广告金融借贷行业，不是通用AI助手，是专业投放工具
2. **可落地建议**：每条建议都附带具体操作步骤，不是正确的废话
3. **行业基准量化**：内置件均分档基准库（超低/低/中/高），诊断有对比参照
4. **多轮对话交互**：像老搭档一样，越聊越懂你，支持追问和约束补充
5. **反馈闭环设计**：用户行为→飞书数据回流→产品持续迭代
6. **完整Portfolio**：PRD→系统提示词→知识库→前端开发→测试验证，完整呈现AI PM工作流程
